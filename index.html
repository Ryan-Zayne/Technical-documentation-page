Introduction to Python

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.
It is used for but not limited to web development(server-side), software development, mathematics, system scripting.

What can Python do?

  • Python can be used on a server to create web applications.
  • Python can be used alongside software to create workflows.
  • Python can connect to database systems. It can also read and modify files.
  • Python can be used to handle big data and perform complex mathematics.
  • Python can be used for rapid prototyping, or for production-ready software development.
As of now, the most recent major version of Python is Python 3. However, Python 2, although not being updated with anything other than security updates, is still quite popular.
It is also apparent to note that Python can be written in a text editor as well as in Integrated Development Environments, such as Thonny, Pycharm, Netbeans or Eclipse which are particularly useful when managing larger collections of Python files.

Syntax

Python syntax can be directly executed in the Command Line as so: >>> print("Hello, World!") Hello, World!
However though, the Python syntax places a huge importance on the indentation of codes as it uses that to indicate blocks of code. This very unlike other programmming language that only recommend indentation for mere readibilty of overall code.
An example of this is shown as follows: if 5 > 2:   print("Five is greater than two!")
If the above indentation was skipped, the output would be a glaring error message in the form of "Syntax error".
The number of spaces is up to the programmer but the recommended for optimum readability is four spaces or in other words, one tab. But for the code to actually work, it has to be at least one space.

Variables

Variables are containers for storing data values. It's important to note that Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.
An example of this is: x = 5 y = "John" print(x) print(y)
Variables do not need to be declared with any particular type, and can even change type after they have been set.

Datatypes

In programming, data type is an important concept.
Variables can store data of different types, and different types can do different things. Python has the following data types built-in by default, in these categories:

Text Type: str Numeric Types: int, float, complex Sequence Types: list, tuple, range Set Types: set, frozenset Boolean Type: bool Binary Types: bytes, bytearray, memoryview None Type: None

Loops

Python has two primitive loop commands:

  • While loops
  • For loops

While loops

With the while loop, we can execute a set of statements as long as a condition is true.
Example: Print i as long as i is less than 6: i = 1 while i < 6:     print(i)     i += 1
Note: remember to increment i(e.g: i += 1), or else the loop will continue forever.
The while loop requires relevant variables to be ready, in this example we need to define an indexing variable, i, which we set to 1.

For loops

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).
This is less like the for keyword in other programming languages, and works more like an iterator method as found in other object-orientated programming languages.
With the for loop we can execute a set of statements, once for each item in a list, tuple, set etc.
Example: Print each fruit in a fruit list: fruits = ["apple", "banana", "cherry"] for x in fruits:         print(x)
The for loop does not require an indexing variable to set beforehand.

Functions

A function is a block of code which only runs when it is called.
You can pass data, known as parameters, into a function.
A function can return data as a result.

Creating a function

In Python a function is defined using the def keyword.
An examples shows this in action: def my_function():   print("Hello from a function")

Calling a Function

To call a function, use the function name followed by parenthesis.
Example: def my_function():   print("Hello from a function") my_function()
